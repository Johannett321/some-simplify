# React Spring Starter Template

## Getting started

First fork this repository, then clone your fork. After cloning, open a terminal, move into the project folder and run the init script. The commands look like this:

```
git clone <your_repo>
cd <repo_folder>
./init.sh
```

When choosing a groupid, use two words that are separated by punctuation. A couple of examples are com.enthemed or io.triggme.

When choosing an appname, write the application name in PascalCase with each word starting with a capital letter and with no spaces between them.

## Included technologies

### Frontend

This template includes Vite, React, Typescript, ShadCN, Tailwind, Axios, OpenApi Generator and framer motion.

### Backend

The backend uses Spring Boot together with Spring Security, Spring Data JPA, Stripe, mail support, JWT tokens, OpenAPI and pdfbox.

### Database

The project is configured to use PostgreSQL.

## Other information

The application supports a multi tenant architecture. A tenant can represent an organization, a workspace or any other logical separation that suits your domain.
